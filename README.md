AI-Powered RAG Study Chatbot | Python, Streamlit, FAISS, Gemini AI

 Developed an **AI-powered RAG Study Chatbot** using **Python, Streamlit, FAISS, Sentence Transformers, and Google Gemini**, enabling users to upload PDF study materials and ask natural language questions.
 Implemented **PDF text extraction, text chunking, semantic embedding generation, vector similarity search, and Retrieval-Augmented Generation (RAG)** to provide accurate, context-aware answers from uploaded documents.
 Integrated a **Gemini AI fallback mechanism** to answer questions using external knowledge when relevant information is not found in the uploaded study materials, with a modular and scalable architecture.
